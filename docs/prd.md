# PRD — 나만을 위한 Todo 웹 애플리케이션

무엇을 만들 것인가 (기능, 요구사항)

## 1. 개요

### 제품 목적
개인 생산성 향상을 위한 Todo 웹 애플리케이션. 빠르게 할 일을 추가/관리하는 심플함과, 카테고리·태그·통계 등의 체계적인 관리 기능을 함께 제공한다.

### 대상 사용자
- 단일 사용자 (개인용)
- 배포된 도메인에서 혼자 사용

---

## 2. 기능 요구사항

### 2-1. Todo 관리 (핵심)
- **Todo 생성** — 제목, 설명, 우선순위, 마감일, 카테고리, 태그를 지정하여 생성
- **Todo 조회** — 전체 목록 조회, 필터/검색 조건 적용
- **Todo 수정** — 모든 필드 수정 가능
- **Todo 삭제** — 단건 삭제
- **완료 토글** — 완료/미완료 상태 전환, 완료 시각 자동 기록
- **순서 변경** — 드래그 앤 드롭으로 Todo 순서 직접 조정 (position 기반)

### 2-2. 카테고리
- **카테고리 생성** — 이름 + 커스텀 색상으로 생성
- **카테고리 목록 조회** — 전체 카테고리 목록
- **카테고리 수정** — 이름·색상 수정 가능
- **카테고리 삭제** — 삭제 시 해당 Todo의 category는 null 처리
- **Todo에 카테고리 할당** — Todo 1개당 카테고리 1개

### 2-3. 태그
- **태그 생성** — 이름으로 생성 (Todo 저장 시 없는 태그 자동 생성)
- **태그 목록 조회** — 전체 태그 목록
- **태그 수정** — 이름 수정 가능
- **태그 삭제** — 삭제 시 연결된 Todo-Tag 관계 제거
- **Todo에 태그 할당** — Todo 1개당 태그 N개 (다대다), 이름 기반 find-or-create

### 2-4. 우선순위
- **우선순위 설정** — HIGH / MEDIUM / LOW 중 선택 (기본값: MEDIUM)
- **우선순위 시각화** — 색상 뱃지로 구분 표시

### 2-5. 마감일
- **마감일 설정** — Date picker로 날짜 입력
- **D-day 뱃지** — 마감일까지 남은 일수 표시 (D-3, D-Day, D+1 등)

### 2-6. 필터 & 검색
- **카테고리 필터** — 특정 카테고리의 Todo만 표시
- **태그 필터** — 특정 태그가 붙은 Todo만 표시
- **우선순위 필터** — HIGH / MEDIUM / LOW 필터
- **완료 여부 필터** — 완료 / 미완료 / 전체
- **텍스트 검색** — 제목/설명 키워드 검색

### 2-7. 캘린더 뷰
- **월간 캘린더** — 날짜별로 Todo 표시
- **날짜 클릭** — 해당 날짜의 Todo 목록 확인
- **날짜에서 Todo 생성** — 클릭한 날짜가 마감일로 미리 설정된 생성 폼 열림
- **날짜에서 Todo 수정/삭제** — 캘린더에서 바로 수정 및 삭제 가능

### 2-8. 통계 대시보드
- **전체 완료율** — 완료된 Todo / 전체 Todo 비율
- **카테고리별 현황** — 카테고리별 Todo 수 및 완료율
- **이번 주 완료 추이** — 월요일 시작 고정 주간 기준, 요일별 완료 Todo 수 (바차트)
- **이번 달 완료 추이** — 이번 달 1일~말일 기준, 일별 완료 Todo 수 (바차트)

### 2-9. 일간 플래너
- **시간 그리드** — 00:00 ~ 24:00 (10분 단위 스냅) 시간축을 화면 왼쪽에 표시. FullCalendar timeGridDay 기반.
- **태스크 풀** — 선택된 날짜의 미배정(시간 미지정) Todo를 화면 오른쪽 목록으로 표시
- **드래그 앤 드롭** — 태스크 풀에서 시간 그리드로 드래그하면 시작/종료 시각 할당 (기본 30분)
- **이벤트 이동/리사이즈** — 그리드 이벤트 카드를 드래그하여 시간 변경, 하단 핸들로 종료 시각 조정
- **시간 해제** — PlannerScheduleModal에서 삭제 버튼으로 스케줄 해제
- **날짜 이동** — 이전/다음 날 이동 버튼
- **완료 체크** — 그리드 이벤트 카드에서 완료 체크 시 Todo의 `completed` 함께 업데이트
- **스케줄 영속성** — 시간 할당 정보는 `TodoSchedule` 별도 테이블에 저장 (Todo 엔티티 비침습)
- **모바일 탭 모드** — 1280px 미만에서 '일정'/'할 일' 탭으로 전환하여 표시. 탭 모드에서는 DnD 비활성화.

## 3. 비기능 요구사항

- **인증** — 조회(GET)는 누구나 가능. 생성/수정/삭제(CUD)는 단일 비밀번호 로그인 필요.
- **반응형** — 모바일/데스크탑 레이아웃 모두 대응
- **다크 모드** — 라이트/다크 테마 토글 지원 (next-themes)
